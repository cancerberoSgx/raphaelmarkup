test using text and font.

we create text using the <text element. If not font attribute is passed to text, the text will be created with paper.text(...) if a font name is passed in the font attribute, then the shape is created with paper.print() and:

1) the font script should be added and registered ni raphael
2) attributes size, origin and letter-spacing can be passed:
size -  size of the font, default is 16
origin - could be "baseline" or "middle", default is "middle"
letter_spacing - number in range -1..1, default is 0

Also the text string can be passed via the text attribute or in the CDATA (inside the element)
