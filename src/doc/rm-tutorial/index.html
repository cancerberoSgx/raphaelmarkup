raphaëlMarkup tutorial

@author: Sebastián Gurin

Here I will show you how to draw in your web pages using XML. We will go from a very simple use case to the most advanced features, example by example living in this page itself.

Getting started

First of all download and uncompress the latest version of raphaëlMarkup. from there we will need the following files

  • rm-min.js - raphaëlMarkup javascript file
  • deps/raphael-min.js - a javascript library needed by raphaël - the drawing library in which raphaelMarkup is based.
  • deps/jquery-1.7.1.min.js - a javascript library needed by raphaëlMarkup

Now create a new directory named "drawing1" and put these three files inside. Also create a new file named drawing1.html with the following content

<!DOCTYPE HTML>
<html>
<head>
	<meta charset="UTF-8"/>
	<title>drawing 1</title>
	<script src="jquery-1.7.1.min.js"></script>
	<script src="raphael-min.js"></script>
	<script type="text/javascript" src="rm-min.js"></script>
</head>
<body>

<table>
<tr>
<td>And at the right, there is my first drawing using raphaelMarkup: 
</td>

<td id="example1Container">

	<raphael style="display:none">
	<paper width="400" height="400" container-id="example1Container">
	<circle cx="200" cy="200" radius="160" fill="red"></circle>
	<text x="50%" y="50%" font-size="18" fill="brown">A circle and some text
	not so much...
	</text>
	</paper>
	</raphael>

</td>
</tr>
</table>

<script>
rm.render();
</script>
</body>
</html>

you can open this exact example and for viewing it with your favourite browser here.
