raphaëlMarkup tutorial

@author: Sebastián Gurin

Here I will show you how to draw in your web pages using XML. We will go from a very simple use case to the most advanced features, example by example living in this page itself.

Introduction

It is recommended that the user know the basics of XML language and a little javascript.

Getting started

First of all download and uncompress the latest version of raphaëlMarkup. from there we will need the following files

  • src/rm-min.js - raphaëlMarkup javascript file
  • src/deps/raphael-min.js - a javascript library needed by raphaël - the drawing library in which raphaelMarkup is based.
  • src/deps/jquery-1.7.1.min.js - a javascript library needed by raphaëlMarkup

Now create a new directory named "drawing1" and put these three files inside. Also create a new file named drawing1.html in the same directory with the following content

<!DOCTYPE HTML>
<html>
<head>
	<meta charset="UTF-8"/>
	<title>drawing 1</title>
	<script src="jquery-1.7.1.min.js"></script>
	<script src="raphael-min.js"></script>
	<script type="text/javascript" src="rm-min.js"></script>
</head>
<body>

<table>
<tr>
<td>And at the right, there is my first drawing using raphaelMarkup: 
</td>

<td id="example1Container">

	<raphael>
		<paper width="400" height="400" container-id="example1Container">
			<circle cx="200" cy="200" radius="160" fill="red"></circle>
			<text x="50%" y="50%" font-size="18" fill="brown">
			A circle and some text
			not so much...
			</text>
		</paper>
	</raphael>

</td>
</tr>
</table>

<script>
$(document).ready(functino(){
	rm.render();
})
</script>
</body>
</html>

you can open this exact example and for viewing it with your favourite browser here.

Congratullations you just drawed your first shapes in your html document using raphaelMarkup. The important things to notice here are:

  • this is a simple html page that shows a table with some text at first column and a drawing in the second column.
  • Inside the second <td> we put a <raphael> element. with a <paper inside. The paper references the parent html element in which the paper must be rendered, with 
    container-id="example1Container"
  • At last there is a script that calls rm.render() when the document is loaded. This calls the raphaelMarkup render engine and renders each <raphael> element found in the

Inside the paper we draw two shapes, a red circle and a brown text. Try for example, changing the color of the circle editing fill="yellow" and refresh the html page for seing the changes.

For simplicity In the following examples, we will only give you the code under the <raphael> tag. More information about how to inserting your XML drawing in a html web page can be found in Using raphaelMarkup examples

It is also noticeable that this document it self s an example of an html page with raphaelMarkup XML drawings in it.

Simple shapes and attributes

In the following example we draw some shapes setting some of its attributes both explicitly in the XML element and using CSS like syntax

<raphael>
<paper width="500" height="400">
	<circle cx="100" cy="100" radius="50" fill="red" stroke-width="20" stroke="blue"></circle>
	<ellipse cx="100" cy="100" rx="80" ry="20" fill="yellow" opacity="0.5"></ellipse>
	<path stroke-width="5" stroke-color="brown" fill="green" opacity="0.2">
	m141.42508,88.60652l-19.93359,-70.83847l-41.15945,-6.60992l17.75754,140.41547l-89.66449,-87.96709l12,103l28,77.00002l75,-63.00002l45,-58l-27,-34z</path>
	<text x="150" y="120" transform="r45" font-size="22">strange and easy</text>
</paper>
</raphael>
m141.42508,88.60652l-19.93359,-70.83847l-41.15945,-6.60992l17.75754,140.41547l-89.66449,-87.96709l12,103l28,77.00002l75,-63.00002l45,-58l-27,-34z ...strange and easy...

Things to notice here:

  • We define a paper with four shapes inside, a circle, an ellipse a path and a text.
  • Shapes are printed in order, newer in front of older. In this example, the text appears in the front while the circle appears at the back of them all.
